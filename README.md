# streamloots

Streamloots Card API

## Quick Start

Get started developing...

```shell
# install deps
npm install

# run in development mode
npm run dev
```

## Try It
* Open you're browser to [http://localhost:3000](http://localhost:3000)
* Invoke the `/cards` endpoint 
  ```shell
  curl http://localhost:3000/api/v1/cards
  ```
